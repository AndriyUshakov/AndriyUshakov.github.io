Stripping down Samsung Stock ROM
unbranded international Galaxy Note 3 (SM-N9005, KitKat 4.4.2, N9005XXUGNG1)

Kept Packages / Removed Packages / Deleted Files

What can be done if you not using many pre-installed apps/features? The "extra" features take a storage space, memory and consume some battery energy. Possibly, they can be a source of security/privacy issues.

The adjusting system is a highly individual process. My list of kept and removed packages is shown below. No boot loops, no any forced closes (application crashes) have been noted.

The packages are sorted by name of apk file. The chosen format is following:

  • [Title of application] - [Name of apk file with full path] - [Package name]

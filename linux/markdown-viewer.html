Markdown Viewer

To open local Markdown files in Firefox (Debian 9) with Markdown Viewer (v.1.3.0), you need to create a new MIME type.

~/.local/share/mime/packages/text-markdown.xml:

Then run:

$ update-mime-database ~/.local/share/mime.

Xorg

Autostarting Application

The system autostart directory is /etc/xdg/autostart/. The user autostart directory is $HOME/.config/autostart/. By placing an application's .desktop file in one of the Autostart directories the application will be automatically launched during startup of the user's desktop environment after the user has logged in. [Desktop Application Autostart Specification]

Configure Second Monitor

  1. Check names of monitors:
    $ xrandr -q
  2. Adjust "positions":
    $ xrandr --output VGA-1 --right-of LVDS-1

Set Brightness

$ xrandr --output LVDS-1 --brightness 0.7

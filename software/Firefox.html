Security/Privacy

uMatrix filters net requests. Development has ended.

uBlocko is a general-purpose blocker.

CanvasBlocker blocks canvas-API to prevent canvas fingerprinting.

ClearURLs removes tracking elements from URLs. Source

CSS Exfil Protection sanitizes and blocks any CSS rules which may be designed to steal data. Source

LocalCDN emulates remote frameworks (e.g. jQuery, Bootstrap, AngularJS) and delivers them as local resource. Source

Privacy Possum wrenches common commercial tracking methods by reducing and falsifying the data gathered by tracking companies. Source

Misc

Bookmark Dupes displays/removes duplicate bookmarks. Source

Selective Bookmarks Export Tool exports selected bookmarks. Source

Feedbro is advanced feed reader (RSS, Atom, RDF) with built-in social media plugin support (Youtube, Facebook, Twitter, Instagram and etc.).

Print Edit WE helps to choose the contents of a web page prior to printing.

Translate Web Pages translates whole page using Google or Yandex or selected text using Google, Yandex, Bing or DeepL. Source

Configuration

The large number of browser settings can be configured via user.js. For the desktops, create your user.js somewhere and drop it into your Profile directory. The apply user.js in unrooted Android is more difficult (see, arkenfox Wiki).

! uBlocko (1.16.4.10): a general-purpose blocker.

! NoScript (5.1.8.4) protects against cross-site scripting attacks (XSS), cross-zone DNS rebinding / CSRF attacks (router hacking) and Clickjacking attempts.

✘ No Resource URI Leak adds option to deny resource:// access to Web content.

✔ Cookies Exterminator deletes cookies and localStorage objects as soon as they become unused.

! Decentraleyes (1.4.2) - local emulation of Content Delivery Networks.

! Google search link fix (1.5.4) prevents Google and Yandex search pages from modifying search result links (Note: it doesn't work if JavaScript is disabled).

! Session Manager (0.8.1.13) saves and restores the state of all or some windows.

! ExportHTMLFolder exports Bookmarks folder in HTML.

✘ Bookmark Deduplicator removes duplicate bookmarks.

✔ Advanced Night Mode provides three customizable schemes to invert colors in browser for comfortable surfing at night.

✔ Moon Tester Tool is designed for testing compatibility of add-ons with Pale Moon.

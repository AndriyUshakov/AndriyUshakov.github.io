IceCat

GNU IceCat is the GNU version of the Firefox browser. IceCat is based on the Firefox Extended Support Release (ESR) with removal of trademarked artwork and proprietary components [IceCat Wiki].

user.js

The large number of browser settings can be configured via user.js. Create your user.js somewhere and drop it into your Profile directory.

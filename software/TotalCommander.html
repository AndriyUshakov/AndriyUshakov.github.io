Compare Files with WinMerge (MS Windows)

To compare two files (in left and right panels) by help of WinMerge, add a new button to the button bar or add a new item to the Menu Start:

Command: "C:\Program Files (x86)\WinMerge\WinMergeU.exe"
Parameters: %P%N %T%r
Icon file: C:\Program Files (x86)\WinMerge\WinMergeU.exe

PPS-Sim

Polarized Positron Source - Simulation (PPS-Sim) is a cross-platform, open-source simulation tool for development of a (polarized) positron source.

PPS-Sim is based on

① Geant4 - a toolkit for the simulation of the passage of particles through matter,

② ROOT - a modular scientific software toolkit mainly written in C++,

③ Qt - a cross-platform application framework and widget toolkit for creating graphical user interfaces.

Other Tools

FLUKA - a fully integrated particle physics MonteCarlo simulation package.

ANSYS - a finite element analysis software used to simulate engineering problems.
